# coffee_shop_website
☕ Sipology Coffee Shop Website

Sipology is a modern and responsive coffee shop website designed using HTML, CSS, and Bootstrap. It provides an attractive user interface to showcase coffee menus, services, and contact details.

🔹 Features
Responsive design for all devices
Attractive homepage with images and sliders
Separate pages: Home, About, Services, Menu, Contact
Stylish UI with modern layout
Easy navigation with sticky navbar
🔹 Technologies Used
HTML5
CSS3
Bootstrap 5
🔹 Purpose

This project is created to demonstrate front-end web development skills and design a user-friendly coffee shop website.
